# Portfolio Website

## Live Demo
The website can be viewed live at https://sbabych.vercel.app/

A personal portfolio website created using HTML5, CSS3, JavaScript, and Tailwind CSS.

## Features

- Fully responsive design
- Animated sections
- Smooth scrolling
- Dropdown menu
- Email contact form
- Projects section displaying images and links to live demos and GitHub repos
- Social media links

## Technologies

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- EmailJS
